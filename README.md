## Features

- **Navigation Header** - Sticky on scroll with blurry background and hamburger menu for mobile view (JavaScript)
- **Hero Section** - Background image with large headline and 2 Call-to-Action buttons
- **Information Section** - 3-4 cards displaying key information
- **Pricing Section** - Three price tiers: Sample, Premium, and Enterprise
- **Footer** - Navigation links for easy site navigation
- **Smooth Scroll** - Link anchor scrolling for better user experience

- HTML5

## How to Run Locally

1. **Clone this repository:**
```bash
   git clone https://github.com/sereitepy/aditi-module-1.git
```

2. **Navigate to the project folder:**
```bash
   cd aditi-module-1
```

3. **Open in VS Code or any code editor:**
```bash
   code .
```

4. **Install the Live Server extension:**
   - Search for "Live Server" by Ritwick Dey in VS Code extensions
   - Install it

5. **Run the project:**
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - The site will open in your browser at `http://localhost:5500`
